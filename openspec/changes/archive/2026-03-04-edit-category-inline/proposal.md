## Why

Users currently cannot edit categories directly on the landing page, which adds friction to the category customisation process. Allowing inline editing by clicking on the category text will streamline the user experience and make choosing their preset categories much faster.

## What Changes

- Make the category text on the landing page transaction list clickable.
- Clicking the text will switch the display to an inline edit mode (e.g., a dropdown or text input).
- Saving the new category will update the transaction immediately without requiring a full page reload or navigating away.

## Capabilities

### New Capabilities
- `inline-category-edit`: Inline editing of transaction categories on the landing page

### Modified Capabilities

## Impact

- Landing page transaction list component
- Category update API endpoints or state management
